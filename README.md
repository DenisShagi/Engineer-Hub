# Engineer HUB

```
S - Добавление стикера
Esc - Снять выделение с инструмента
Зажимаем ctrl - можем выделять несколько стикеров одновременно =
Продолжаем выделять через shift
Удалить стикер - клавиша delete или backspace
Двойной клик - переходим в режим редактирования, д
ля сохранения кликаем вне области стикера либо нажимаем esc
```
